Pokemon Team Builder

In this project, I wanted to explore using a recommendation algorithm that can help users build competetive Pokemon teams. The program takes data from official Pokemon tournaments and uses the data collected on each player's team to filter out which Pokemon are more frequently used in competetive play. The user can then begin to build their own team and the program will make suggestions for Pokemon that may synergize well with the user's current team build.

Featured Skills: Python, A-Priori algorithm, web scraping
